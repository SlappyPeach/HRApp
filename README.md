# HRApp – Система автоматизации кадрового учёта

**HRApp** — это WPF-приложение, предназначенное для автоматизации работы кадровой службы в образовательных учреждениях. 
Система охватывает полный цикл управления персоналом: приём на работу, оформление приказов, ведение документации, табель учёта рабочего времени, напоминания, отчёты и экспорт в Word/Excel.

## Основной функционал

###  Сотрудники

* Учёт сотрудников, отображение и фильтрация.
* Карточка сотрудника по форме Т-2 (все персональные и кадровые данные).
* Трудовая книжка (с сохранением и экспортом).
* Поддержка увольнения с фиксацией даты.
* Поддержка прикрепления резюме.

### Приказы

* Приказы о приёме, переводе, увольнении, отпуске, командировке, больничном.
* Автоматическая генерация номеров приказов (формат `ПРЕФИКС-ГОД-НОМЕР`).
* Экспорт приказов в Word по шаблонам.

### Документы и журналы

* Входящие, исходящие, внутренние документы.
* Журнал приказов и журнал регистрации.

### Табель рабочего времени

* Автоматическое формирование табеля по выбранному месяцу и году.
* Учёт отпусков, больничных, выходных, рабочих дней.
* Итоги по явкам и отклонениям.
* Экспорт табеля в Excel.

### Напоминания

* По срокам отпусков, аттестаций, договоров, удостоверений, повышения квалификации.
* Настраиваемый период (3/7/30 дней).
* Отметка "обработано", фильтрация.
* Экспорт напоминаний в Excel.
* Уведомление при запуске приложения.

### Отчётность

* Приказы, отпуска, больничные, список сотрудников — за период.
* Новый отчёт по гражданству.
* Экспорт всех отчётов в Excel.

### Награды

* Учёт наград сотрудников: дата, номер, тип, подразделение.
* Форма наград с привязкой к сотруднику.
* Экспорт списка наград в Excel.

## Используемые технологии и библиотеки

| Технология                         | Назначение                              |
| ---------------------------------- | --------------------------------------- |
| **.NET 8 / WPF**                   | GUI и основа приложения                 |
| **Entity Framework Core (SQLite)** | Работа с встроенной БД (файл SQLite)    |
| **ClosedXML**                      | Экспорт отчётов и табеля в Excel        |
| **Xceed.DocX**                     | Экспорт документов и приказов в Word    |
| **MaterialDesignInXAML**           | Современный UI, стили, карточки, кнопки |
| **MVVM (частично)**                | Разделение логики и представления       |

## Установка и запуск

1. Скачать проект

2. Открыть в Visual Studio

   * Открыть `HRApp.sln`.

3. Установить NuGet-зависимости

   * `ClosedXML`, `Xceed.Words.NET`, `MaterialDesignThemes`, `EFCore.Sqlite`, `EFCore.Tools`.

4. Настройка базы данных

   * По умолчанию используется `SQLite`. БД создаётся в `%AppData%\HRApp\hrdatabase.db`.
   * Миграции EF Core используются для обновления схемы.

5. Запустить проект

   * Нажать `F5` или выбрать `Запуск без отладки`.
